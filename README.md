# TinyKeys
A Free MPC Expansion Pack - Tiny but Versatile Multisampled Music Instruments (Keygroups)

## How to install?
Copy the complete TinyKeys folder to your MPC (folder Expansions).

## Which instruments?
* Accordion
* Bass
* Brass
* EGuitar
* EPiano
* Flute
* GlideLead
* Grand
* Harmonica
* Marimba

* Organ888
